# Reshit

Reshit is a modern AI-powered Jewish platform designed to make Jewish knowledge, prayer, halacha, learning and community tools more accessible through modern technology and intelligent user experience.

Built for the digital generation, Reshit combines artificial intelligence, Jewish content, modern design and smart digital systems into a unified platform focused on accessibility, simplicity and innovation.

---

## Vision

Reshit was created with the goal of building a bridge between ancient Jewish tradition and the modern technological world.

The platform focuses on creating a smarter, more accessible and modern Jewish digital experience while preserving the depth, authenticity and values of traditional Jewish content.

---

## Core Features

- AI-powered Jewish assistant
- Smart prayer and siddur experience
- Jewish calendar and zmanim
- Torah, Halacha and Talmud tools
- Hebrew RTL-first interface
- Personalized user experience
- Community and social features
- Modern mobile-first design
- Cross-platform accessibility

---

## Technology

Reshit is built using modern technologies and AI-driven development workflows, including:

- React
- TypeScript
- Tailwind CSS
- Vite
- Mobile-first architecture
- AI integrations
- Modern UI/UX systems

Reshit aims to modernize the digital Jewish ecosystem by providing:
